# Chess board component

A chess board web component

## Component reference

Usage

```
<loloof64-chessboard
        size="222"
    ></loloof64-chessboard>
```

### Properties

| Name | Role                                              | Type   | Default |
|------|---------------------------------------------------|--------|---------|
| size | Common size (width/height) of the board in pixels | number | 100.0   |
